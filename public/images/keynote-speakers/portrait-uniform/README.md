# Uniform keynote speaker portraits

Created on 14 July 2026 from the source photographs in `../portrait-hq/`.

All eight files are:

- 1200 x 1500 pixels
- 4:5 portrait format
- transparent PNGs
- consistently framed as centered conference-speaker portraits

The built-in OpenAI image editor was used for identity-preserving background
extraction. Each prompt required the exact face, age, expression, hair,
clothing, pose, skin tone, and body proportions to remain unchanged while only
the background and portrait framing were standardized. A flat chroma-key field
was then converted to transparency locally, with a contracted soft matte and
edge-spill cleanup.

The original source photographs remain unchanged in `../portrait-hq/`.
